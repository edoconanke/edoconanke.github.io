Gradient Descent Method to Solve the Equation f(x) = 0

2019-11-18 · 6 mins to read

The codes listed in this post have been tested in Python 3.6 and Matplotlib 3.1.1.

In my previous post, I explained how to use Newton’s method to solve the equation $f(x) = 0$. In this post, I’ll introduce another widely used method in the numerical calculation - gradient descent.

Actually, the gradient descent method is usually to find the minimum of the evaluated function $f(x)$. The following figure shows the core idea behind the gradient descent method.

Example: solve the equation $\cos(x) = x$.

Although the gradient descent method is used to find the local minimum of the function $f(x)$. We can not use it directly to solve the equation $f(x) = 0$. But finding the root of $f(x) = 0$ is equivalent to finding the minimum of $|f(x)|^2$.

We use the equation $\cos(x) = x$ as a example. We define the function $f(x) = \cos(x) - x$. To find the root of $f(x) = 0$, we only need to find the minimum value of $|f(x)|^2 = (\cos(x) - x)^2 = \cos^2(x) - 2x\cos(x) + x^2$. The following code plots the $|f(x)|^2 = (\cos(x) - x)^2$:

We can write the Python code to solve this problem.

import numpy as np
import matplotlib.pyplot as plt

x = 1.0  # the starting value of x
learning_rate = 0.02  # learning rate
iteration = 200
error = 1e-5  # convergence error

x_history = [x]

for i in range(iteration):
    f_grad = -2.0*(np.cos(x) - x)*(np.sin(x) + 1)
    #lr_x = lr_x + x_grad**2
    #x = x - learning_rate/np.sqrt(lr_x)*x_grad
    x = x - learning_rate*f_grad
    x_history.append(x)

    print("interation ", i, ": x = ", '%.8f' % x, ", gradient = ", f_grad)
    if(abs(f_grad) < error):
        break

$x$ value of each step is stored in the list variable x_history. If the gradient of a certain $x$ is less than error, the iteration is stopped and the $x$ is considered to be the solution. The output of the code above is:
